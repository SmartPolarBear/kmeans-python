# kmeans-python

Homework for 2022 XMU pattern recognition lesson. A implementation of k-means clustering algorithm based on Pandas and
NumPy.

## Run With

### Environment

The code of this project is run with

- Python 3.10
- NumPy 1.21.5
- Pandas 1.4.1

## Usage

This project mainly works on the Iris dataset. To run this project, run main.py with the following arguments:

| Argument  | Description                          |
|-----------|--------------------------------------|
| --dataset | The path to Iris dataset, a CSV file |
